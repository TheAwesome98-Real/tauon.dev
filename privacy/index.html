privacy

by using certain services that i have created, some of your data may be stored. here they are:

github pages (tauon.dev)

this site is hosted on github pages. you can find 2 pieces of information on how github handles your data when you visit sites hosted on github pages here and here.

lucerne (lucerne.tauon.dev)

when you visit any page on this site, your IP address, user agent, and the file you accessed, are logged. this is the default behaviour of nginx, and it will stay enabled for me to diagnose if i am being attacked.

wiki (wiki.tauon.dev)

while the wiki is nonfunctional, the information about lucerne.tauon.dev also applies to it.

Motion JPEG experiments (mjpeg.tauon.dev)

the information for lucerne applies, due to the servers for the Motion JPEG experiments being reverse-proxied through nginx.

additionally, a hashed version of your IP is stored in an in-memory database, in order to determine which functions you have accessed on each experiment.

the 88x31s have been hidden because you have reduced motion turned on, and there are many fast flashing gifs. there is no way to unhide them at this time, but i will try to add that in future.
